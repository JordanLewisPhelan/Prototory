## Using Obsidian to view the files as I see them

Once you have downloaded and installed [obsidian](https://obsidian.md/). 

- Pull my github repo down and save it somewhere temporarily.
- Start by opening the application and you will see not much, just a few options. 
- Click on `Open folder as Vault` - Vaults are basically the folder locations for the markdown files as they contain some custom viewing scripts so are better to be contained to their own folders.
- Go to wherever you have my project saved on your device and navigate to `Protory/Autumn Additions & Progress`, .obsidian should be there and just click `open folder` and that should open the Vault where all the markdown files will be organised as well as any images I used should be labelled here too.
